## Setup

```
./spsim_d -s
```

## Run

```
./spsim_d -r
```

With refreshed data
```
./spsim_d -rr
```

## Test

```
./spsim_d -t
```

## Format

```
./spsim_d -f
```

## Styling instructions

1. To add new dependencies use:

```
uv add <package>
```

2. Always use types.

3. Write tests for each function you add.

4. Try to keep functions small and focused and avoid side effects.

5. Format code with:

```
ruff format
```
